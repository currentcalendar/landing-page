This template provides a minimal setup to get React working in Vite with HMR and some ESLint rules.

Also Tailwind it is installed for productivity in the development of the landing page

- To install all the packages of the project run:

` npm install`

- To run the application in development mode run:

` npm run dev`

- If you want to see how it fits in mobile devices run the following command and access the IP:port shown in the terminal only if you are connected to the same network as your computer:

` npm run dev -- --host`

You can access the application at http://localhost:5173/
